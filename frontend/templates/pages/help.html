Ayuda y Guías

Todo lo que necesitas saber para sacar el máximo partido a Ollash, tu asistente de IA local.

¿Qué es Ollash?

Ollash es un asistente de inteligencia artificial que funciona completamente en tu ordenador, sin enviar ningún dato a internet. Puedes preguntarle cualquier cosa, pedirle que escriba código, que revise tus proyectos o que te explique conceptos técnicos en lenguaje sencillo.

A diferencia de ChatGPT o GitHub Copilot, todo el procesamiento ocurre localmente gracias a Ollama. Tus conversaciones son privadas, el sistema funciona sin conexión a internet y puedes elegir el modelo que mejor se adapta a tu ordenador.

Primeros pasos

1

Instala Ollama

Descarga Ollama desde ollama.com e instálalo. Después abre una terminal y descarga el modelo recomendado:
ollama pull qwen2.5:4b

2

Inicia Ollash

En la carpeta de Ollash, ejecuta: python run_web.py
Abre tu navegador en http://localhost:5000

3

Escribe tu primera pregunta

Haz clic en Chat en el menú lateral y escribe lo que necesites. Por ejemplo: "¿Qué hace este código Python?" o "Explícame qué es una variable".

4

Crea tu primer proyecto (opcional)

Haz clic en Nuevo Proyecto y sigue el asistente de 3 pasos. Ollash generará la estructura completa de un proyecto de código a partir de tu descripción.

Casos de uso prácticos

📋 Explicar código existente

Explícame qué hace esta función Python: def calcular(n): return n * 2 + 1
El AI explica la lógica línea por línea en lenguaje sencillo, sin tecnicismos innecesarios.

✨ Crear un proyecto desde cero

Crea una aplicación web de lista de tareas con Python y Flask, con base de datos SQLite
Genera todos los archivos: app.py, templates HTML, CSS, requirements.txt y README con instrucciones.

🔍 Revisar y mejorar código

Revisa este código y dime si hay errores o mejoras posibles: [pega tu código aquí]
Recibe sugerencias concretas con ejemplos de código corregido y explicaciones del por qué.

🌐 Aprender conceptos técnicos

¿Qué es una API REST y cómo funciona? Explícalo de forma sencilla con un ejemplo
Una explicación clara con analogías del mundo real, ejemplos de código y pasos a seguir.

✅ Generar tests automáticamente

Genera tests unitarios para esta función: [pega tu función aquí]
Crea un archivo de tests completo listo para ejecutar con pytest, con casos positivos y negativos.

Crear tu primer proyecto

1

Haz clic en "Nuevo Proyecto"

En el menú lateral, bajo Principal, encontrarás el botón "+ Nuevo Proyecto". Se abre un asistente de 3 pasos.

2

Pon nombre y elige una plantilla

Escribe el nombre del proyecto y selecciona el tipo: Default (genérico), FastAPI (API web), React (interfaz web) o Automation (scripts).

3

Describe tu proyecto con detalle

Cuanto más detallada sea la descripción, mejor será el resultado. Por ejemplo: "Una aplicación para gestionar tareas diarias con Python. Debe tener login de usuario, base de datos SQLite, y una interfaz web simple con Flask."

4

Revisa el resultado en "Mis Proyectos"

Ollash genera todos los archivos en tiempo real (puedes ver el progreso en pantalla). Una vez terminado, ve a Mis Proyectos, selecciona el tuyo y explora la estructura en Archivos.

Preguntas frecuentes

¿Mis conversaciones se envían a internet?

No. Todo el procesamiento ocurre en tu ordenador mediante Ollama. Ningún dato sale de tu máquina. Puedes usar Ollash incluso sin conexión a internet (una vez descargado el modelo).

¿Qué modelo debo usar?

Equilibrado (4B) es el más recomendado para uso diario: buena calidad y velocidad razonable. Rápido (0.8B) responde en segundos pero con menos precisión. Potente (30B) es ideal para proyectos complejos pero requiere una GPU con bastante VRAM.

El AI da respuestas incorrectas o con errores, ¿qué hago?

Los modelos locales pequeños cometen errores. Si recibes código incorrecto, pide una revisión directamente en el chat: "Hay un error en el código anterior, el problema es que...". También puedes probar con un modelo más grande en Configuración.

¿Por qué aparece el indicador "Ollama offline"?

Ollama debe estar ejecutándose en tu ordenador. Abre una terminal y ejecuta ollama serve. El indicador verde en la barra lateral o en la cabecera confirmará la conexión correcta.

¿Puedo usar mis propios modelos de Ollama?

Sí. Descarga cualquier modelo compatible con Ollama (ollama pull nombre-modelo) y luego ve a Configuración → Avanzado para asignarlo. Cualquier modelo del catálogo de Ollama es compatible.

¿Cómo funciona "Mis Proyectos"?

Muestra todos los proyectos que Ollash ha generado, almacenados en la carpeta ./generated_projects del servidor. Selecciona uno para ver su estructura de archivos, historial de versiones Git y documentación generada automáticamente.

¿Cuánta RAM necesito?

El modelo 0.8B necesita ~1 GB de RAM. El 4B necesita ~4-5 GB. El 7B necesita ~8 GB. El 30B necesita ~20 GB y se recomienda GPU dedicada. Si tu ordenador tiene poca RAM, usa el modelo Rápido (0.8B).

¿Cómo cambio el tema visual (claro/oscuro)?

Haz clic en el icono de sol/luna en la barra superior, o ve a Configuración → General y selecciona el tema que prefieras. La preferencia se guarda automáticamente.

Guía de Configuración

La mayoría de usuarios no necesitan cambiar nada. Sin embargo, si quieres ajustar el comportamiento:

  • URL de Ollama — Cambia si tienes Ollama en otro ordenador de tu red. Por defecto: http://localhost:11434
  • Modelo principal — Elige según la RAM disponible y la velocidad que necesitas.
  • Tema — Claro u oscuro según tu preferencia.

Para opciones avanzadas (roles de modelo, flags de funcionalidades, nivel de sandbox, parámetros del LLM), ve a la pestaña Avanzado en la página de Configuración. Estas opciones están pensadas para usuarios técnicos.
