To pass a specific array of models to `benchmark.py`, you can use the `--models` argument when running the script from the command line.

For example:

```bash
python benchmark.py --models model1 model2 model3
```

-   `model1`, `model2`, `model3` would be the names of the Ollama models you wish to benchmark.
-   If you do not provide the `--models` argument, the script will, by default, benchmark all locally available Ollama models.
